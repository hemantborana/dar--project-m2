# MongoDB vs Column-Family (Cassandra)
MongoDB is document based.
Cassandra stores rows inside column-families.
MongoDB supports dynamic fields.
Cassandra best for huge write throughput.
Both scale horizontally, but Cassandra has no JOIN-like operations.
